Nastavení hodnot formulářových polí

Toto je archivovaná verze české komunitní stránky o CMS Drupal. Nejde na ni již přidávat nový obsah. Pro aktuální informace a kontakt s komunitou navštivte Drupal.cz
Vložil/a X-Frox, 8 Říjen, 2007 - 21:07
Kategorie: Jak na to?
Týká se verze:

Potřeboval bych poradit. Vytvářim modul a mám získanou strukturu (pole) pro formulář, kterou zbývá nyní jen vyrenderovat přes drupal_render. Nepodařilo se mi však do této struktury vložit nějaké defaultní hodnoty jendotlivých polí. Například nastavená hodnota #value u textarea, která by se měla normálně do prvku podle theme funkce vložit, není ve vyrenderovaném formuláři nastavena. Tak se ptám, jeslti někdo nevíte, čím to je, nebo jak tuto hodnotu nastavit. A jak ji pak ještě nastavit například u typu File (zkoušel jsem to přes #atributes přidat hodnotu value, ale rovnež bez účinku).

Protoze hodnota #value je

Napsal uživatel Jakub Suchý dne 8 Říjen, 2007 - 21:24.

Protoze hodnota #value je predkonfigurovana hodnota, ktera se NEDA zmenit. Pokud uzivatel do formulare cokoliv doda, drupal to zmeni na #value.

Spravne je pouzit #default_value

Pozor na bezpecnostni rozdil: zatimco #default_value automaticky prochazi pres XSS parser a je to tedy bezpecna hodnota, #value takovym parserem neprochazi. Pokud tedy do #value davate hodnotu od uzivatele (cokoliv z $_GET, $_POST, $_COOKIE, $_SERVER, $form_value), tak ji vzdy MUSITE prohnat pres check_plain() ci prislusnou formatovaci funkci.

File podle me nastavit nejde. Mozna to najdete obecne pro HTML, ale mam za to, ze to prohlizece neumi…

// hlavny formular pre upravu a vkladanie udajov pracoviska function pracovisko_form($data = NULL) {   $form = array();    $form['pracovisko'] = array(     '#type' => 'fieldset',     '#title' => t('Department data'),   );   $form['pracovisko']['pracovisko'] = array(     '#type' => 'textfield',     '#title' => t('Department ID'),     '#default_value' => $data->pracovisko,     '#required' => TRUE,   );   // atd.    return $form; }

Cize ked som potreboval mat prazdny formular zavolal som drupal_get_form(pracovisko_form()) a ked som to chcel mat naplnene datami zabalil som to do funkcie ktora najprv vytiahla data do premennej $data → drupal_get_form(pracovisko_form($data)).

Inak som to volal cez hook_menu():

// menu pre pridanie pracoviska $items[] = array(   'path' => 'admin/fem/pracoviska/add',   'title' => t('Add FEM department'),   'callback' => 'drupal_get_form',   'callback arguments' => array('pracovisko_form_add'),   'access' => $admin_access,   'type' => MENU_LOCAL_TASK, );

a funkcia:

/**  * Spracovanie formulara pre pridanie noveho pracoviska  */ function pracovisko_form_add() {   $form = pracovisko_form();   return $form; }
