Skrývání TinyMCE

Toto je archivovaná verze české komunitní stránky o CMS Drupal. Nejde na ni již přidávat nový obsah. Pro aktuální informace a kontakt s komunitou navštivte Drupal.cz
Vložil/a miloso, 11 Březen, 2008 - 11:00

Používáte-li z jakýchkoli důvodů WYSIWYG editor TinyMCE, možná jste již narazili na situaci (a zejména, pokud Vašemu webu vládne CCK), kdy TinyMCE potřebujete pro určité textové políčko vypnout. Pojďme si ukázat, jak na to – je to velmi snadné.

Nejprve si do schránky zkopírujte celou funkci theme_tinymce_theme ze souboru sites/all/modules/tinymce/tinymce.module. Otevřete si soubor sites/all/themes/vaše_téma/template.php (je-li Váš web oděn do hávu dodaného přímo s Drupalem, vřele doporučuji používané téma překopírovat právě do adresáře sites/all/themes, aby nebyly Vámi provedené změny zahozeny při příští aktualizaci Drupalu) a vložte do něj obsah schránky. V těle funkce si povšimněte prvního bloku příkazů case, pod poslední příkaz tohoto bloku (tedy ještě před řádek s příkazem unset, více v ukázce) vložte nový řádek, na němž bude ležet opět příkaz case a jeho argumentem bude název formulářového políčka, u něhož chcete zobrazení TinyMCE potlačit.

function epc_tinymce_theme($init, $textarea_name, $theme_name, $is_running) {   switch ($textarea_name) {   …     case 'description': // taxonomy terms     case 'field-adresa-0-0-value':       unset($init);       break;   …   }   … }

Jak zjistit jméno formulářového políčka? Je-li toto políčko dílem CCK, pak použijte příslušný atribut name ze zdrojového kódu stránky a veškeré speciální znaky jako podtržítka nebo hranaté závorky nahraďte pomlčkami (například z mnou použitého name="field_adresa_0[0][value]" se stane v příkladu předvedené case 'field-adresa-0-0-value'). V opačném případě použijte také hodnotu atributu name, ale žádné nahrazování neprovádějte.
