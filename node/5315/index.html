Krok 1) musí se to nastavit

Napsal uživatel Eva Rázgová dne 14 Červen, 2009 - 22:04.

Krok 1) musí se to nastavit v souboru fckeditor.config.js (obecně viz README soubor modulu FCKeditor, Configuration, bod 8). V profilu, kde to chcete mít, přidáte ‚Style‘, takže například v [„DrupalFiltered“] místo [‚FontFormat‘] budete mít [‚FontFormat‘, ‚Style‘]. Asi to bude chtít vymazat cache. Pak se vám ukáže selectbox se styly v panelu FCKeditoru.

Krok 2) Definovat vlastní styly v souboru fckstyles.xml, který si překopírujete do Vašeho tématu vzhledu a tam ho budete upravovat. Přidáte tam (příklad):

<Style name="Zvyrazneni-1" element="span">  <Attribute name="class" value="zvyrazneni-1" />  </Style>  <Style name="Zvyrazneni-2" element="span">  <Attribute name="class" value="zvyrazneni-2" />  </Style>

krok 3) V css souboru vašeho (!!!) pod/tématu vzhledu zadáte, jak mají vypadat vaše třídy .zvyrazneni-1 a .zvyrazneni-2

krok 4) V adminu Drupalu v konfiguraci FCK editoru v daném profilu nastavíte CSS: Editor CSS: define css > a dole nastavíte cestu k CSS souboru, kde máte ty vaše styly…

Predefined styles: use theme fckstyles.xml (což se pochopitelně odvíjí od bodu 2, v jiných případech to může být jinak).
