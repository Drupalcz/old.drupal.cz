Zdravím, myslím že si

Napsal uživatel SpatnyNick dne 31 Leden, 2007 - 08:38.

Zdravím, myslím že si musíte vytvořit vlastní typ obsahu. Je to modul CCK – jestli ho nemáte, nainstalujte si ho. Pak si v menu Administrace->Typy obsahu->Přidat typ obsahu založte nový typ článek (třeba „Článek s podnadpisem“). Pak si tento článek dejte Upravit a přidejte si nový pole (Textové/Text field), přidejte mu váhu např. na –5 aby se ukazoval nad textem. Nastavte poli parametry, jaké požadujete a pak už je na nastavení stylu tématu, jak se Vám ten text bude zobrazovat…

Pozn.: můžete měnit i původní typ článek, nemusíte vytvářet nový…

Že to jako pitomý vypisuje i názvy těch polí do textu?

no to ano, můžeš si upravit style.css tématu který používáš a nastavit něco jako:

.node .content .field-field-test .field-label {
display: none; /* tohle ti zakreje titulek položky */ } místo „.field-field-test“ si doplň jméno typu který si vytvoříš, viz HTML zdroj nějkaýho zkušebního nodu toho typu kterej si vytvoříš.

no a druhá varianta (a asi i čistší) která mě napadá je udělat si vlastní proceduru která ti ten node vyrenderuje – zkopíruj si soubor node.tpl.php na node-test.tpl.php a v něm si můžeš uplně překopat vzhled (renderování) node typu „test“ (viz název souboru) – myslím že ten node máš v proměnné $node, takže si tam přidej někde na začátek do php tagů „print_r($node);“ zobraz si ten node a koukni do html stránky – pak uvidíš strukturu, kde v tom objektu $node co je uložený.. pak už je to spíš o html než php
