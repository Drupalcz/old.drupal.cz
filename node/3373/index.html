re: Dva bloky nad obsahem a pravým sidebarem

Napsal uživatel Anonymous (bez ověření) dne 13 Červenec, 2008 - 19:20.

Díky za příspěvky, ale stále jen hladově kroužím, možná těsně před cílem. Modul Panels můj problém neřeší, nemá takovou šablonu rozložení, jakou potřebuji.

Dle rady Igorika jsem si vytvořil nový region, který se mi ale nedaří umístit nad hlavní sekci a pravý panel.

Stavím na thematu Bluebreeze a nový region se snažím umístit do page.tpl.php:

<?php if ($header): ?>       <div id="header-region">         <?php print $header ?>       </div>     <?php endif; ?>    </div>      <div id="container" class="<?php if ($sidebar_left) { print "withleft"; } if ($sidebar_right) { print " withright"; }?> clear-block">     <div id="main-wrapper">      <div id="main" class="clear-block">        <!--</div><br /> -->       <?php if ($mission): ?><div id="mission"><?php print $mission ?></div><?php endif; ?>       <?php if ($top_story):?><div id="top-story"><?php print $top_story ?></div><?php endif; ?>       <?php if ($content_top):?><div id="content-top"><?php print $content_top ?></div><?php endif; ?>       <?php if ($title): ?><h1 class="title"><?php print $title ?></h1><?php endif; ?>       <?php if ($tabs): ?><div class="tabs"><?php print $tabs ?></div><?php endif; ?>       <?php print $help ?>       <?php print $messages ?>       <?php print $content ?>       <?php if ($content_bottom): ?><div id="content-bottom"><?php print $content_bottom ?></div><?php endif; ?>     </div>     </div>      <?php if ($sidebar_left): ?>       <div id="sidebar-left" class="sidebar">         <?php print $search_box ?>         <?php print $sidebar_left ?>       </div>     <?php endif; ?>      <?php if ($sidebar_right): ?>       <div id="sidebar-right" class="sidebar">         <?php print $sidebar_right ?>       </div>     <?php endif; ?>    </div>

Nový region jsem nazval top_story. Daří se mi jej dát buďto stejně jako top_ content, což je nad hlavním obsahem mezi sidebary, nebo nad sidebary a hlavní obsah. Tuším, že zádrhel spočívá v té klauzuli s print withleft a print withright, ale netuším, co s tím.
