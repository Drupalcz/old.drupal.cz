Volání externí drupal funkce ve vlastní funkci (v template.php)

Toto je archivovaná verze české komunitní stránky o CMS Drupal. Nejde na ni již přidávat nový obsah. Pro aktuální informace a kontakt s komunitou navštivte Drupal.cz
Vložil/a mat., 16 Říjen, 2007 - 12:14
Kategorie: Jak na to?
Týká se verze:

Ahoj mam konkretni problem pri vytvareni vlastni funkce. S customizaci Drupalu se teprve seznamuju, ale blokl jsem se uz tady..

template.php

function zen_mujTest() {   drupal_add_js ('$(document).ready(function(){$("p.jsTest").fadeIn(10000);}); ','inline');   $output = '<p class="jsTest" style="background-color: lightyellow; ">This is an example of an effect which is built into the core jQuery library...';   return $output; }

do page.tpl.php pak dam

<?php print zen_mujTest() ?>

ale mam problem s tim, ze jinak normalne fungujici funkce drupal_add_js se v te me funkci zen_mujTest() neprovede.

(Funkce drupal_add_js nedela nic jineho nez „prevede“ tuto syntaxi a vyvola pozadovany javascript)

Pokud mou funkci doplnim o vlozeni javascriptu natvrdo

<script type="text/javascript">$(document).ready(function(){$("p.jsTest").fadeOut(3000);}); </script>

taxe se pozadovana operace provede..

Takze cele to trci na tom, ze chci vyuzit funkci drupal_add_js, ale dle vyse uvedeneho mi jeji volani uvnitr me funkce nefunguje… Mozna k tomu pristupu od zacatku blbe, volat takto v mé funkci nejakou dalsi.

Muzete nekdo poradit plis?

(jedna se o hratky s jquery)
