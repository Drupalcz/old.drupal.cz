pokud jsou to vsechno nody

Napsal uživatel karlos dne 16 Duben, 2010 - 14:05.

pokud jsou to vsechno nody tak se to samozrejme nema podle ceho generovat(nepozna to tu logickou strukturu, to by slo jenom u taxonomy treba). mějme: Domů :: O nas :: team :: vedeni :: popis

  • O nas = node/1
  • team = cesta-k-view (pro ilustraci at nejsou same nody)
  • vedeni = node/2
  • popis je aktualne zobrazena stranka
takze cesty bude 
 node/1
 cesta-k-view
 node/2
a titles budou 
 O nas
 Team
 Vedeni

dulezite je ze kazdy „drobecek“ je dan cestou a titulkem(proto to musi byt presne jak pisu vcetne zlomu radku!). tyto dva elementy musi byt na stejnem radku v tech dvou textareach. Domů je tam generovano automaticky. posledni element(aktualni title) tam dostanete upravou template.php

function phptemplate_breadcrumb($breadcrumb) {   if (!empty($breadcrumb)) { // uncomment the next line to enable current page in the breadcrumb trail //    $breadcrumb[] = drupal_get_title();     return implode(' :: ', $breadcrumb);   } }
