dva formuláře

Napsal uživatel Vojtěch Kusý dne 25 Srpen, 2008 - 20:06.

Pokud chceš dva formuláře v JS tabech, tak je vytvoř pomocí webformů a JS Tools / Tabs. Jak jsem naznačoval v příspěvku „JS taby se dělají celkem“. Do těch ‚#content‘ se vyrenderuje formulář. Jelikož ten formulář je v případě webformu vlastně node, vyrenderování by mělo být celkem jednoduchý. Pomocí node_load se ten formulář načte z databáze a pomocí node_view se vyrenderuje do HTML.

Kromě Webformu si nainstaluj ještě modul JSTools, je nutné zapnout submodul Tabs.

Tento kód se vloží do stránky (typ „page“) se zapnutým PHP filtrem.

<?php  $form = array();  $form['formulare'] = array(   '#type' => 'tabset', );  // Tab 1 - Osoba, 4 znamena nid formulare $form_node = node_load(array('nid' => 4)); $rendered_form = node_view($form_node);  $form['formulare']['tab1'] = array(   '#type' => 'tabpage',   '#title' => t('Personal address'),   '#content' => $rendered_form, );  // Tab 2 - Firma, 5 znamena nid formulare $form_node = node_load(array('nid' => 5)); $rendered_form = node_view($form_node);  $form['formulare']['tab2'] = array(   '#type' => 'tabpage',   '#title' => t('Firm address'),   '#content' => $rendered_form, );  print tabs_render($form);  ?>

Vymýšlel jsem to z hlavy, není to odzkoušeno, ale mělo/mohlo by to fungovat.
