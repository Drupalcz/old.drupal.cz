Takže závěr po dni lopocení:

Napsal uživatel agir dne 1 Únor, 2011 - 17:51.

Takže závěr po dni lopocení: CCK Computed filed je taky nanic, protože ve Views se to nehlásí jako pole typu Datum, takže se tam nenabídne možnost filtrovat v definovaném úseku datumů. Nakonec jsem ale na to našel poměrně dobrý způsob. Modul se jmenuje: Views PHP Filter. Tím vám přibude ve filtrech nový filtr. Je to vlastně textové pole, kam si napíšete vlastní SQL dotaz. Pokud by tedy někdo někdy potřeboval podobné filtrování, tak níže posílám obsah mého filtru. Enjoy!;-)

Je třeba použít modul: Views

Napsal uživatel agir dne 1 Únor, 2011 - 17:54.

Je třeba použít modul: Views PHP Filter
Pak ve Views: Filtr > uzly: PHP Filtr

Název CCK pole:

*Typ obsahu: narozeniny
*Název CCK pole: field_narozeni­ny_date

Název CCK pole v databáze, kde je uloženo datum narozenin je:

*Tabulka: content_type_na­rozeniny
*Sloupec:node_da­ta_field_naro­zeniny_date
Obsah filtru:

 

$start_from_today = "-10"; //Offset -10 dnů od aktuálního data $end_from_today = "30"; //Offset +30 dnů od aktuálního data $nodeids = array(); // Vytvoří pole, které bude posláno zpět filtru  /* Provádím join, abych si spojil node a jeho CCK pole. STR_TO_DATE() Rozseká text do datumových polí podle zadané šablony DATE_FORMAT() Z poslaného datumu vybere podle vzoru jen požadovanou část CURDATE( ) Aktuální datum (2010-10-01) DATE_ADD(  '2010-10-01', INTERVAL 30 DAY ) Připočte či odečto od poslaného data daný počet dnů Pozor na formát data dne, je třeba zadávat: %%d a ne %d */ $query = "SELECT nid FROM (SELECT node.nid AS nid,node.title AS node_title,node_data_field_narozeniny_date.field_narozeniny_date_value AS node_data_field_narozeniny_date_field_narozeniny_date_value,node.type AS node_type,node.vid AS node_vid FROM node node LEFT JOIN content_type_narozeniny node_data_field_narozeniny_date ON node.vid = node_data_field_narozeniny_date.vid  WHERE DATE_FORMAT(STR_TO_DATE(node_data_field_narozeniny_date.field_narozeniny_date_value, '%Y-%m-%%dT%T'), '%m-%%d') >= DATE_FORMAT(DATE_ADD( CURDATE( ) , INTERVAL $start_from_today DAY ), '%m-%%d') AND DATE_FORMAT(STR_TO_DATE(node_data_field_narozeniny_date.field_narozeniny_date_value, '%Y-%m-%%dT%T'), '%m-%%d') <= DATE_FORMAT(DATE_ADD( CURDATE( ) , INTERVAL $end_from_today DAY ), '%m-%%d') ) AS narozky";  $my_result = db_query($query);  //Uloží dotaz do proměnné  //$nodeids[1] = -1; Toto zajistí, abyste pokud nebudou žádný výsledku z databázového dotazu, nedostali chybu z poslání prázdného pole. Nějak mi to nefungovalo - smazal jsem. while ($my_row = db_fetch_array($my_result))  // Vloží čísla uzlů do pole { $nodeids[] = $my_row['nid']; } return $nodeids; // Pošle čísla nodů filtru
