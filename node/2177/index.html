No polozky z views sa

Napsal uživatel SonnY dne 17 Prosinec, 2007 - 09:54.

No polozky z views sa automaticky radia pod seba, ale nie je problem ich ulozit vedla seba. Staci im cez css zadefinovat parameter display: inline. To je to najmenej. Ale co sa sablony tyka, tak v podstate pre teba bude asi najlahsie, ak si vytvoris zvlastnu sablonu a do nej si vlozis premenne, ktore najdes cez contemplates a usporiadas si to v sablone tak ako potrebujes. Alebo to mozes spravit tak ze si vyvoris novy typ obsahhu,vytvoris mu tie polia ktore tam chces(pre tie obrazky) a cez contemplate si to usporiadas a len to cele nastylujes cez css. Cize mas na vyber: bud budes robit vlastnu sablonu a css, alebo len css. Myslim ze najlahsie je usporiadat si to cez contemplate, pridat si tam nejake divy a nastylovat.

Jasne ze ide:

// funkcia l() vygeneruje odkaz podla parametrov // je lepsie pouzit funkciu l() ako priame html kvoli moznosti zmeny cesty print l(   theme('imagecache', 'tvojimagecachetyp', $node->field_tvojobrazok[0]['filepath'])), // vygeneruje html s nahladom obrazku   $node->field_tvojobrazok[0]['filepath'], // cesta k originalnej velkosti obrazku   array('class' => 'lightbox'), // pre vygenerovany odkaz `<a ...>` prida class pre lightbox (pripadne taky ako treba)   NULL, // co je za ? v url   NULL, // co je za # v url   FALSE,   TRUE // v prvej polozke je HTML (inak urobi 'htmlencode') ); }

Daj si modul devel a nechaj

Napsal uživatel Havran dne 17 Prosinec, 2007 - 17:45.

Daj si modul devel a nechaj si vypisat (dev load/dev render nad kazdym node) ako su tie CCK premenne naplnene a podla toho rob ten skript. Ta funkcia l() je vlastne nahrada priameho pisania html.

<div class="field-item">   <?php foreach ((array)$node->field_obr_body as $item) { ?>      <a href="/files/images/<?php print $item['view'] ?>.jpg" rel="lightbox[autobazar]" title=""><img src="/files/images/<?php print $item['view'] ?>.thumbnail.jpg" /></a>   <?php } ?> </div>

Cize miesto <a href...>..</a> ju treba napisat (a spravnymi premennymi naplnat. A samozrejme si skontrolovat zdrojak ci to robi to co clovek chce.
