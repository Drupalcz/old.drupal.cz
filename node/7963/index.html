hook_form_alter, cck a group

Toto je archivovaná verze české komunitní stránky o CMS Drupal. Nejde na ni již přidávat nový obsah. Pro aktuální informace a kontakt s komunitou navštivte Drupal.cz

Kategorie: Jak na to?
Týká se verze: Drupal 6.x

Prosím potřebovala bych poradit s hook_form_al­ter(), který má v node form změnit label v cck poli, které je součátí group.

Můj kód v mujmodul.module vypadá takhle:

function mujmodul_form_alter(&$form, $form_state, $form_id) {   if ($form_id == 'jmenotypuobsahu_node_form') {     global $user;     $myvalue = $user->name;     $form['field_nazevpole'][0]['#title'] .= ' '.check_plain($myvalue);    } }

To funguje, ovšem pokud to pole není ve skupině.

Pro skupinu jsem podle tohoto návodu a komentáře http://drupal.org/node/726282#… přidala název skupiny:

$form[‚group_na­zevskupiny‘][‚fi­eld_nazevpole‘][0][‚#ti­tle‘] .= ' '.check_plain($my­value);

… ale to už nefunguje (neudělá to nic). Víte někdo, jak na to? Předem díky.

Řešení je na světě

Napsal uživatel Eva Rázgová dne 5 Srpen, 2010 - 15:11.

… opět moc děkuju Martinovi Hrabovčínovi a zjistila jsem to, jak jinak, opět v hospodě, tudíž choďte do (Drupal) hospod, ať se něco dozvíte :-)))

mujmodul.module:

function MUJMODUL_form_alter(&$form, $form_state, $form_id) {   if ($form_id == 'NAZEVTYPUOBSAHU_node_form') {     $form['#after_build'][] = 'MUJMODUL_after_build';   } }  function MUJMODUL_after_build($form, &$form_state) {   global $user; //to je jen příklad, budu pracovat s uživatelem   $myvalue = $user->name; //vytáhnu jméno uživatele   $form['group_NAZEVSKUPINY']['field_NAZEVPOLE']['value']['#title'] .= check_plain($myvalue); //přidám jméno uživatele k názvu pole   //dpm($form);    //pomůže mi vyznat se v celém formuláři, vyžaduje zapnutý modul Devel   return $form; }

P.S. karlosi, Tvoje intuice, že tam chybí [‚value‘], byla taky správná a díky ;-)

P.S. pro ostatní: nebude tam ale [‚value‘] vždy, někdy tam může být [‚nid‘], jindy [‚email‘] – záleží, co je to za pole (princip úplně stejný, jako když si třeba píšete default value do CCK polí v PHP). Když nevím, použiju dpm($form).
