Je ta struktura menu

Napsal uživatel Havran dne 6 Leden, 2008 - 11:05.

Je ta struktura menu jednorazovka (urobis a viac sa k tomu nevratis) alebo sa to meni casto? Podla mna si si nevybral dobry nastroj – taxonomy je taxonomy – sluzi na zobrazovanie (a triedenie) skupin, nie na menu. Urcite je jednoduchsie vytvorit si strukturu cez taxonomy ako v menu ale nesplna to ten isty ucel – ked teda chces mat na konci na polozkach zavesenu uz len jednu stranku.

Bol jeden modul – tusim menusplit alebo tak nejak, ktory dokazal rozdelit menu na hlavnu cast a podcasti ktore sa zobrazovali. Skusal som ho ale mne nevyhovoval tak som si nakoniec urobil vlastne riesenie. Nie je sice velmi efektne lebo donho treba natvrdo zapisat menu id a v sablone volat jednu funkciu ktora zobrazi zakladne menu, ale pre moje ucely staci.

Dam to sem aby z toho nieco

Napsal uživatel Havran dne 6 Leden, 2008 - 13:34.

Dam to sem aby z toho nieco mali aj ostatni.

Priklad: Chceme strukturu menu:

Auta   Skoda   Fiat   Mercedes Motorky   Kawasaki   Pionier Traktory   Zetor     Typ 1       Podtyp 1       Podtyp 2     Typ 2

V Drupale v menu administracii si vytvorime nove menu (napriklad Hlavne menu) a donho vytvorime pozadovanu strukturu. To Hlavne menu ma nejake menu id – napriklad 99. Chceme dosiahnut aby sa nam zobrazovali niekde len hlavne polozky (Auta,Motorky,Trak­tory) a ostatne polozky len ked na konkretnu polozku klikneme (napr. klikneme na Traktory a zobrazi sa nam blok s menu polozkami – konkretne Zetor vratane jeho podpoloziek).

Aby to fungovalo urobime si modul (Drupal 5). Najprv .info subor

; $Id$ ; meno suboru: moje_menu.info name = "Moje menu" description = "Moje menu." project = "moje_menu" version = "5.x-1.x-dev"

Potom modul

<?php // $Id$ // meno suboru: moje_menu.module  /**  * Implementation of hook_block().  */ function moje_menu_block($op = 'list', $delta = 0, $edit = array()) {   // bloky budeme zobrazovat len z tohoto podmenu   // ak by ich bolo treba viacej pridame $mid do pola   $my_blocks = array(99);    if ($op == 'list') {     $count = 0;     $menu = menu_get_menu();      // vytvorime blok pre kazdu polozku menu prvej urovne     foreach ($my_blocks as $pid) {       foreach ($menu['visible'][$pid]['children'] as $mid) {         $blocks[$mid.'-'.$count] = array(           'info' => $menu['visible'][$mid]['title'],           'weight' => -5,           'enabled' => 1,           'region' => 'sidebar_left',         );         $count++;       }     }     return $blocks;   }   else if ($op == 'view') {     $delta_array = explode('-',$delta);      // blok vygenerujeme a zobrazime iba pre aktualnu polozku menu     if (!$block = myRootSubMenu($delta_array[0])) {       return;     }      return $block;   } } // */  // funkcia pre zobrazenie hlavnych poloziek menu // tato funkcia sa vola v teme na mieste kde chceme zobrazit hlavne polozky // ako parameter sa pouziva $mid ktory je v hook_block v poli $my_blocks function myRootMenu($pid) {   $flag_first = true;   $menu = menu_get_menu();   $trail = _menu_get_active_trail();    foreach ($menu['visible'][$pid]['children'] as $mid) {     $item = menu_get_item($mid);     $link_item = $item;     $link = '';      while ($link_item['type'] & MENU_LINKS_TO_PARENT) {       $link_item = menu_get_item($link_item['pid']);     }      // prvej polozke pridame class first     if ($flag_first) {       $output .= '<li class="leaf first">';       $flag_first = FALSE;     }     else {       $output .= '<li class="leaf">';     }      if ($trail[1] == $mid && $link_item['path'] != $_GET['q']) {       $output .= l($item['title'], $link_item['path'], !empty($item['description']) ? array('title' => $item['description'], 'class' => 'active') : array('class' => 'active'), isset($item['query']) ? $item['query'] : NULL);     }     else {       $output .= l($item['title'], $link_item['path'], !empty($item['description']) ? array('title' => $item['description']) : array(), isset($item['query']) ? $item['query'] : NULL);     }     $output .= '</li>';   }    return $output; }  // funkcia pre vygnerovanie bloku z menu // tato funkcia generuje samotny block function myRootSubMenu($mid = NULL) {   $menu = menu_get_menu();   $trail = _menu_get_active_trail();    if ($mid == $trail[1]) {     if (($trail[0] == 91 || $trail[0] == 87) && $menu_str = theme('menu_tree', $trail[1])) {       $block['subject'] = $menu['visible'][$trail[1]]['title'];       $block['content'] = $menu_str;     }     return $block;   }    return; }

No a ako je pisane v zdrojovom kode – v sablone stranky na mieste kde chceme zobrazit pozadovane menu (skladajuce sa len z hlavnych poloziek) zavolame funkciu napr. takto:

<?php print '<ul id="IDHlavneMenu1">'.myRootMenu(99).'</ul>'; ?>

Potom ked klikneme na polozku tymto kodom vygenerovaneho menu sa nam zobrazi v blokoch automaticky blok s polozkami podmenu.
