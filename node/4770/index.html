System je jednoduchy: za

Napsal uživatel ivanjaros.net (bez ověření) dne 5 Březen, 2009 - 13:07.

System je jednoduchy: za vytvorenie nodu urciteho typu sa strhne jedne kredit. Taktiez za zvolenie niektorych cck poli daneho nodu(su to vlastne inzeraty s moznostou zvyraznenia a zvyhodnenia pozicie). Kredity ma user v cck poli svojho content profilu.

UP: ja nemam problem s kreditmi ako takymi(jasne som to napisal v prvom poste), ale so zakazom vytvorenia nodov urciteho typu, ak user nema kredit. Tu problem nehladaj. Porblem sa tyka ciso user_access() a jeho obidenia/zrusenia, pokial user nema kredit.
