Hoj. Řešim zobrazování obsazenosti pokojů pomocí modulu Availability Calendars.Funguje dobře. Ke každýmu pokoji přidám kalendář kde zatrhnu obsazený dny. Pomocí views mám vytvořenou stránku s filtrem co mi vypíše podle vloženýho rozsahu dnů volný pokoje.

Potřebuju ale stránku, kde budu mít přepínač měsíců a následnej výpis všech pokojů i s kalendářema pro vybranej měsíc.

Psal jsem issue a dostal jsem odpověď:
In the old (non-field based) version you could show a specific month (or set of months) with URL's served by the availability calendars module. However, since it is a field , this does not seem to be logical anymore, so the feature disappeared. but this old feature would not have helped you anyway, as that was only for 1 calendar.

What you can do is hook into the themeing of the calendars and change the settings that define which period will be shown. The theme that you should preprocess is the availability_ca­lendar_months theme. The theme code details exactly what variables are involved:

<?php function theme_availabi­lity_calendar_mon­ths($variables) {
$year = &$variables[‚y­ear‘];
$month = &$variables[‚mon­th‘];
$months_to_render = $variables[‚set­tings‘][‚show_n­umber_of_months‘];
… } ?> The code just above that contains a preprocess hook example:

<?php function availability_ca­lendar_prepro­cess_availabi­lity_calendar_mon­ths(&$variables) { } ?> This preprocess hook loads the availability. If your hook executes earlier, you can change the year, month and number of months to get what you are asking for. But ensure that it only executes if your view is running, not on other pages…
