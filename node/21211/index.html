Kategorie: Jak na to?
Týká se verze: Drupal 7.x

Zdravím vás,

potreboval by som poradiť ohľadom modulu Simple Pin Map. Modul som nainštaloval vytvoril som si 2 miesta na mape označil ich. Samo mi automaticky vytvorilo tie názvy Košice / Bratislava. Prvá vec skúšal som ich pomocou CSS posunúť trochu ďalej od mapky ale nejak to nešlo inak som spokojný z ich umiestnením. Ten hlavný problém je ten text pod nimi. Read more je link na ďalší page, ktorému nastavím cestu (to viem). Ale ten text „Array“ hľadal som všade možne ale neviem nájsť miesto kde by som mohol vložiť svoj text namiesto toho Array.

Vopred vďaka za rady.
