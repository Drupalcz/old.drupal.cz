Vytvoří se view v podobě:

Napsal uživatel ankerek dne 26 Leden, 2011 - 16:06.

Vytvoří se view v podobě:

Relationships:
Uzly: Votes (nebude se nic nastavovat) // pro tvoření políček a třídění hlasů

Arguments:
Uzly: Nid – Provide default arg. – Node ID from URL // ukáže hlasy ze stránky, na které se nacházíme Fields:

Hlasy: uživatel
Hlasy: hodnota

Sort criteria:
Hlasy: Časové razítko (desc)

Teď to zobrazí seznam hodnocení v podobě: ID uživatele a hodnocení (1 nebo –1) Potřebujeme ve tvaru: jméno uživatele a palec nahoru/dolů Můžeme to udělat přepsáním výstupů jednotlivých polí:

V menu view klikneme na „Theme: Information“ a vidíme políčka a názvy souborů.

Vybereme nějaký název pole s ID uživatele (pozor, aby se nevybralo obecný – views-view-field.tpl.php, které přepíše i ostatní pole). Vytvoříme nový soubor s vybraným názvem a do něj vložíme: <?php $user = user_load($row->{$field->field_alias});
$userpath = ‚user/‘ . $user->uid;
print l($user->name, $userpath); ?>

To se postará o jméno uživatele místo ID.

Nyní se vytvoří soubor druhého pole (hodnota hlasu) a do něj se vloží: <?php if ($output>0) {print ' ‚.$output.‘ ';} if ($output<0) {print ' ‚.$output.‘ ';} ?> Kladný hlas bude mít třídu „thumbs-u“ a záporný „thumbs-d“, potom nastylujete ty třídy s pozadím palce dolů/nahorů.

Soubory se nahrají do složky tématu a poté se klikne na Rescan template files (v Theme: Information).

Můj výsledek je na adrese http://www.muzz.cz/…t/pink-floyd (v pravém bloku).
