No ked sa nahodi modul

Napsal uživatel Havran dne 19 Srpen, 2007 - 21:00.

No ked sa nahodi modul content templates umozni sa pohrat z akymkolvek typom obsahu – zaramovat vystup do tabulky a podobne. To PHP je dobre vediet len na tolko aby clovek rozoznal co je premenna a co obashuje. Priklad pre typ obsahu host ktory mam na stranke www.parcon.sk – takto vyzera jeho template:

<div class="field field-type-text field-field-meno">   <h3 class="field-label">Meno</h3>   <div class="field-items">       <div class="field-item"><?php print $field_meno[0]['view'] ?></div>   </div> </div>  <div class="field field-type-text field-field-priezvisko">   <h3 class="field-label">Priezvisko</h3>   <div class="field-items">       <div class="field-item"><?php print $field_priezvisko[0]['view'] ?></div>   </div> </div>  <div class="field field-type-image field-field-fotografia">   <h3 class="field-label">Fotografia</h3>   <div class="field-items">       <div class="field-item"><?php print $field_fotografia[0]['view'] ?></div>   </div> </div>  <div class="field field-type-text field-field-url">   <h3 class="field-label">Domovská stránka</h3>   <div class="field-items">       <div class="field-item"><?php print $field_url[0]['view'] ?></div>   </div> </div>  <div class="field field-type-text field-field-vystpenia">   <h3 class="field-label">Vystúpenie</h3>   <div class="field-items">     <?php foreach ((array)$field_vystpenia as $item) { ?>       <div class="field-item"><?php print $item['view'] ?></div>     <?php } ?>   </div> </div>  <div class="field field-type-nodereference field-field-link-na-prednku">   <h3 class="field-label">Link na prednášku</h3>   <div class="field-items">     <?php foreach ((array)$field_link_na_prednku as $item) { ?>       <div class="field-item"><?php print $item['view'] ?></div>     <?php } ?>   </div> </div>  <div class="field field-type-text field-field-text">   <h3 class="field-label">Text</h3>   <div class="field-items">       <div class="field-item"><?php print $field_text[0]['view'] ?></div>   </div> </div>
