tak a este jedna zaujimava

Napsal uživatel Peter.tomascik dne 22 Srpen, 2007 - 13:53.

tak a este jedna zaujimava vec s tokenom, token vyuziva aj pathauto. zistil somn ze naprik tomu ze mam v path auto nadefinovane pri node cestu v tvare [type]/[title] tak mi pri vytvoreni akehokolvek typu node vytvara cestu v tvare audio/[title]
napr pri editacii node typu hra mi namiesto cestu hra/nazov-hry vytvori castu audio/nazov-hry
pozeral som a problemy mam s path auto, ato nodetitle, a oba moduly potrebuju k svojej cinnosti prave modul token.
akym sposobom mozem token uplne odinstalovat, a co vsetko musim v DB zmazat aby sa zmazal uplne a mohol som ho znova instalnut?

tak som tento porblem

Napsal uživatel Peter.tomascik dne 23 Srpen, 2007 - 10:01.

tak som tento porblem vyriesil. cely problem sposobuje zapnuty modul audio, ktory s nejakeho dovodu zblbne tokeny, a tym padom nefunguje korektne ani path auto, ani auto nodetitle. jednoducho niektore tokeny sa prestanu vypisovat tak ako som uvadzal hore. akonahle sa modul audio vypne, problemy zmiznu.

bol by niekto schopny tento problem supnut na drupal.org ja to zo svojou lamanou anglictinou asi nezvladnem.

este raz vdaka vsetkym za support.
