Dobrý den,
obracím se na zkušené Drupalisty s prosbou o pomoc. Na práci s css používám Firebug, kde si dohledám class nebo id a podle toho upravuji nebo přidávám styly jednotlivých částí. Ale neumím si poradit v Drupalu s dvěma situacemi:


1. – pokud prvek nemá class nebo id. Jak se toto řeší? Jak mu ho přidělit?
2. – změnit pořadí prvků. např. (jméno → příspěvek) → (příspěvek → jméno). Vím jak toho jednoduše docílit přímo v modulu, ale také vím, že se to tak dělat nemá. Dohledal jsem, že se to řeší nejspíše pomocí template, např: guestbook-entry.tpl.php, ale vůbec nevím jak by měl vypadat konkrétní zápis. Asi něco jako v comment.tpl.php:

mnozstvo modulov ma nejaky.tpl.php subor (napr. spominany comment.tpl.php) – ten skopiruj do adresara s vlastnou temou. tam ho mozes upravit ako len chces.
