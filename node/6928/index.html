Ahoj všem dobrým lidem. Už chvíli si hraju s modulem Printer, e-mail and PDF versions 6.x-1.10. Chtěl jsem udělat šablonu na tisk jenže se mě nedaří přidat jiný obsah do tiskového výstupu než je v orig šabloně.
Dle: For a more fine-grained customization, it is possible to use a print[_format][­.node-node-type].tpl.php file located in either the current theme or the module directory (format is either html, mail or pdf, and node-type is Drupal's node type (e.g. page, story, etc.)).

Obsah orig šablony je: 

<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN" "http://www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd">  <html xmlns="http://www.w3.org/1999/xhtml" lang="<?php print $print['language']; ?>" xml:lang="<?php print $print['language']; ?>">    <head>      <?php print $print['head']; ?>      <title><?php print $print['title']; ?></title>      <?php print $print['scripts']; ?>     <?php print $print['robots_meta']; ?>     <?php print $print['base_href']; ?>     <?php print $print['favicon']; ?>     <?php print $print['css']; ?>   </head>   <body<?php print $print['sendtoprinter']; ?>>     <?php if (!empty($print['message'])) {       print '<div class="print-message">'. $print['message'] .'</div><p />';     } ?>     <div class="print-logo"><?php print $print['logo']; ?></div>     <div class="print-site_name"><?php print $print['site_name']; ?></div>     <div class="print-header"><?php print $print['header']; ?></div>     <p />     <div class="print-breadcrumb"><?php print $print['breadcrumb']; ?></div>     <hr class="print-hr" />     <h1 class="print-title"><?php print $print['title']; ?></h1>     <div class="print-submitted"><?php print $print['submitted']; ?></div>     <div class="print-created"><?php print $print['created']; ?></div>     <div class="print-content"><?php print $print['content']; ?></div>     <div class="print-taxonomy"><?php print $print['taxonomy']; ?></div>     <div class="print-footer"><?php print $print['footer_message']; ?></div>     <hr class="print-hr" />     <div class="print-source_url"><?php print $print['source_url']; ?></div>     <div class="print-links"><?php print $print['pfp_links']; ?></div>   </body> </html>

Používám Adaptivetheme a z něj sem chtěl vložit header-region.
 tedy:

<div class="print-header-region"><?php print $print['header-region']; ?></div>

Bohužel nefunguje. Jo a abych nezapoměl tuto šablonu mám vloženou do adr. modulu. Zkoušel jsem to i do themes ale tam si to modul nevzal. Nevím proč…
Má s tím ted někdo zkušenosti? Diky Radek

Napsal uživatel Radek Pliska dne 1 Únor, 2010 - 14:20.

Pokud to bude někdy někdo potřebovat. Možná ten kod není úplně ok ale funguje.

<?php global $language; $language->language;  $hlavicka = node_load(22); $hlavicka_lang = node_load(22);  $header = node_load(24); $header_lang = node_load(24);  $kopfzeile = node_load(24); $kopfzeile_lang = node_load(24);  if ( $language->language == $hlavicka_lang->language) { print '<span class="print_header">'. $hlavicka->body .'</span>'; } elseif( $language->language == $header_lang->language) { print '<span class="print_header">'. $header->body .'</span>'; } else print '<span class="print_header">'. $kopfzeile->body .'</span>'; ?>

Jak je výše uvedeno, vytvořil jsem si pro hlavičku i patičku node. Přeloži ale vložil už jen NID node_load(<bol­d>XX</bold>).

Jen pozor pokud používáte subtheme. Pokud budete chtít uložit šablonu do themes adresáře, tak to musí být jen v subtheme a to v jeho kořenovém adr. (aspon ja to mam otestovano v adaptive theme). To mám odzkoušeno jako základní název print.tpl.php.

Jen jsem zatím nepřišel na to jak nastylovat výstup pro PDF. Vzhledem že používám TCPDF tak to ani nejde aspon podle stránek modulu print http://drupal.org/project/print

Pokud víte někdo řešení dejte prosím vědět. Díky
