podla vyjadrenia merlina, autora views to nejde spravit. http://drupal.org/node/122261

merlinofchaos – April 1, 2008 – 15:18 Status: active » won't fix There are problems with checkboxes that make them ill suited to exposed filters due to the interaction of checkboxes and GET.

  1. When using GET arguments, each checkbox has its own name. Thus, while using a select box you'll get „http://www.e­xample.com?fo­o=1,2,3“ with checkboxes the URL becomes „http://www.e­xample.com?fo­o[1]=1&foo[2]=1&fo­o[3]=1“.
  2. 1) isn't that bad, but this one is: If no boxes are checked, FAPI will fill the checkboxes to the ‚default‘ state rather than having no checkboxes filled, meaning you cannot have checkboxes defaulting to on. This is because if there are no checkboxes checked, nothing is sent in the URL; if nothing is sent, fapi doesn't see anything.

Because of these problems, I've decided to generally not uses checkboxes in place of select with exposed filters.

pre drupal 5/views1 existoval modul views_checkboxes, resp. dalsie, sam som nejaky poupravovany pouzival, ale pre views2 nic momentalne nie je. skusim sa pohrat s tou temou ci nieco vymyslim.

Unset defaultnich hodnot

Napsal uživatel Okca (bez ověření) dne 3 Duben, 2009 - 14:30.

Problem vidim v nastaveni defaultnich hodnot checkboxu pri volani funkce form_builder a nasledne form_set_value, kdyz je sestavovan view exposed form. „nezaskrtnute“ hodnoty checkboxu je nutne odstranit, aby se nasledne nepridavali do view query jako dalsi filtrovaci podminka.

Je tedy nejenom potreba vymenit typ form elementu select na checkboxes, ale i pridat kontrolu form_state[values], coz lze napriklad pomoci #after_build funkce.

Priklad:

/** * Hook form_alter pro views_exposed_form */ function MODULE_form_views_exposed_form_alter(&$form, $form_state) {   $form['nejaky_prvek_ktery_je_select']['#type'] = 'checkboxes';   $form['nejaky_prvek_ktery_je_select']['#after_build'][] = 'MODULE_checkboxes_after_build'; }  /**  * Vlastni after build funkce  * Defaultni hodnota pro nezvolene checkboxy je (int) 0  */ function MODULE_checkboxes_after_build($form, &$form_state) {   foreach($form_state['values'][$form['#name']] as $key => $value) {     if (!$value) {       unset($form_state['values'][$form['#name']][$key]);     }   }   return $form; }

Snad to pomuze ;o) Drupalu zdar.
