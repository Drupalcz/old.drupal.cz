Otvor si

Napsal uživatel SonnY dne 17 Duben, 2008 - 21:17.

Otvor si fckeditor.con­fig.js. Tam mas vsetky tieto profily a kazdy tento profil ma aktivne rozne funkcie. Skus sa s tym ked tak pohrat, pripadne vytvorit vlastny profil. Tiez si skontroluj ci mas najnovsiu verziu modulu a spravnu verziu fck editoru:

The 6.x-1.x release of fckeditor.module is fully compatible with FCKeditor 2.5.1+. The 5.x-2.x release of fckeditor.module is fully compatible with FCKeditor 2.5.1+, however it should also work with the 2.4.x versions of the FCKeditor. The 5.x-1.2 release of fckeditor.module works with the 2.4.x versions of the FCKeditor component. The 4.x release of fckeditor.module works with the 2.4.x versions of the FCKeditor component.
