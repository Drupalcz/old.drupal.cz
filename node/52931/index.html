Toto je archivovaná verze české komunitní stránky o CMS Drupal. Nejde na ni již přidávat nový obsah. Pro aktuální informace a kontakt s komunitou navštivte Drupal.cz
Vložil/a mmizur, 24 Leden, 2016 - 13:18
Kategorie: Jak na to?
Týká se verze: Drupal 7.x

Dobrý den, pro nový projekt jsem vytvořil virtuální server na localhost. Nainstaloval Drupal 7, ale instalace modulů případně kontrola update skončí chybou: „Your server does not support installing modules and themes from this interface. Instead, install modules and themes by uploading them directly to the server, as described in the handbook.“

Zkoušel jsem databázi mysql i pgsql, ale v tom chyba není. Stránky se rozjedou, ale nejsem schopen instalovat moduly ze stránek ani kontrolovat aktualizace (vím, že se obojí dá dělat manuálně :-)).

Zřejmě je někde chyba v nastavení Apache pro virtualní web, takže ten nekomunikuje s vnějším světem.

Nevíte někdo co a kde odkomentovat/do­plnit v konfiguraci Apache/PHP nebo je chyba v host?
