Zdravím, může mi někdo poradit jak vytvořit nějaké takové menu, které je na obrázku? Tohle je defaultní nastavení menu v motivu Startup Growth Lite.

Používám motiv Corolla s menu Superfish, které je umístěné v regionu header a pomocí modulu Floating block se mi menu posouvá, ale nevytvoří ten potřebný efekt „overlay“ přes celou šířku stránky.

Předpokládám, že to je nějaký javascript a úprava css, ale nevím co přesně hledat. Případně jakou použít alternativu za Superfish.

takže jsem zkusil ten modul

Napsal uživatel f3nix dne 31 Březen, 2015 - 13:31.

takže jsem zkusil ten modul Sticky Navigation, zvolil jsem motiv a jako selector jsem dal #block-superfish-1, tedy blok s menu, který potřebuji. v custom css motivu corolla jsem nastavil toto: #block-superfish-1{ width: 710px; } menu se zobrazí, ta pevná šířka je proto, aby se mi na webu, který je v ostrém provozu zobrazovali všechny položky menu v jednom řádku, když bych nastavil 100%, tak mi to leze do dvou řádků dále jsem nastavil opět v custom css: .stickynav-active{ position: fixed; top: –10px; width: 100%; background: rgba(255, 0, 0, 0.8); } výsledek je, že menu se posune doprostřed stránky (to ale až tak nevadí) a je vidět i nějaký ten „overlay“, ale jenom na tu fixní šířku 710px a já potřebuju, aby se zobrazoval na celé stránce
