V Drupale sa na toto da

Napsal uživatel Havran dne 20 Duben, 2007 - 20:39.

V Drupale sa na toto da napisat jednoduchy modul. Vid:

http://api.drupal.org/…on/hook_user

Tu je subor sites/all/mojmodul/mojmodul.info:

; $Id$ name = MojModul description = Pokusny modul na ukazku hook_user

Tu je subor sites/all/mojmodul/mojmodul.module:

<?php // moj modul ktory ulozim do 'sites/all/mojmodul/mojmodul.module  // napisem vlastny hook_user function mojmodul_user($op, &$edit, &$account, $category = NULL) {   // vykona sa len pri vkladani noveho pouzivatela (registracii)   if ($op == 'insert') {     drupal_set_message('Vitaj '.$account->name.'!');   } }

To vsetko ulozis do adresara sites/all/mojmodul a normalne aktivujes ako novy modul – mal by sa zobrazit v zozname modulov.

-- Editovane: malo tam byt $account->name a nie $edit->name a este na tom riadku na konci chybala bodkociarka ;. Takto to v pohode slape – vypise Vitaj pouzivatel! kde pouzivatel je meno registrovanej o­soby…

Presne to myslim. Konkretne se jedna Debian GNU/Linux 4.0. Ten prikaz zaregistruje na ejabberd server noveho uzivatele.

Melo by to udelat neco podobneho: 
 $jmeno = pepa  # ziska z drupalu
 makepasswd > $heslo    # vygeneruje nahodne heslo
 $mail = uzivatel@server.cz
 ejabberdctl register $jmeno muj_server.cz $heslo
 echo "Jmeno: " $jmeno "heslo: " $heslo | mail pepa@mail.iq

Takze ako zaklad zoberies

Napsal uživatel Havran dne 20 Duben, 2007 - 22:30.

Takze ako zaklad zoberies ten modul co som napisal a dopises tam tie tvoje prikazy – pomocou niektoreho z tychto prikazov – http://sk.php.net/…ref.exec.php

Ale doporucujem si zobrat na pomoc niekoho skusenejsieho lebo dovolovat pustat na Unixe prikazy cez php a este k tomu takto ako to potrebujes je diera do systemu jak hrom. Treba to poosetrovat.

Ako pisem ten modul co som napisal je na toto presne vhodny zaklad – to ze nevies PHP to nerobi horsim riesenim. Pre Drupal je to najlepsie mozne riesenie.
