O webu

Web byl vytvořen nejenom jako webová prezentace křesťanského společenství v Trutnově, ale slouží též základními informacemi o aktuálním dění (pozvánky apod.). Zajímavostí může být jeho vznik, kdy v širém okolí byl prosazován nápad vytvořit stránky na zelené louce pouze za pomoci html (to se psal rok 2008!). Nebylo tedy snadné přesvědčit rádoby zkušené webové harcovníky o využití redakčního systému, které nikdo neznal. Výsledkem bylo několik mladých nazlobených jedinců, kteří se nemohli realizovat a dát okolí naodiv své znalosti html. Zároveň však původní web na jádru 5.x. Žel administrátor - tedy má maličkost - vystudoval pouze účetní školu a posléze se již věnoval pouze teologii, nebyl schopen upgradovat web na vyšší verzi a tak vše udělal ještě jednou, pěkně od začátku. Nyní je tedy web verze 6.x.
