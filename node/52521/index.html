V konfiguraci pole mám nastavení:

  2. Target type: Node
  3. Mode: Simple (with optional filter by bundle) – na výber zde mám checkboxy všech typů nodů. Zaškrtnul jsem „Box“ to je ten který by se měl referencovat.
  4. Sort by: A property of the base table of the entity

    Sort property: nid

    Sort direction: Descending

  5. Number of values: 1
  6. Required field: On

Publishing options:

Synchronize translations:
Tam také není nic zaškrtnuté
