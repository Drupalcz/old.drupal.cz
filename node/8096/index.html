Svoje tpl.php soubory mají

Napsal uživatel Eva Rázgová dne 25 Srpen, 2010 - 10:09.

Svoje tpl.php soubory mají také jednotlivá pole (ze začátku poradí modul Theme developer).

Nebo můžete zobrazování polí vypnout a vypisovat si všechno sám v node.tpl.php, ale jak správně tušíte, pak se musíte postarat i o výpis labelu atd.

Správně je mít v tpl.php co nejjednodušší zápis a všechny podmínky a funkce dávat do template.php (klíčové slovo: preprocess funkce, preprocessing).

A v neposlední řadě, jakmile to vypisujete v šabloně sám (jakýmkoliv způsobem), musíte se starat taky o bezpečnost (keywords: wriging secure code). Budete potřebovat hlavně check_plain() a check_markup()

P.S. jo a pokud budete chtít custom vypisovat obrázky, asi užijete funkci field_file_load(), která vám načte celý $file podle FID.

Ak chces mat teaser zvlast

Napsal uživatel pointer dne 31 Srpen, 2010 - 16:00.

Ak chces mat teaser zvlast v premennej tak si ho tam musis najprv vytvorit v preprocces funkcii. Ja som pouzil tento postup:

V template.php (template.php je subor kam davas preprocess funkcie a ine funkcie pri praci s temamy, vid manual):

function NAZOV_TVOJEJ_TEMY_preprocess_node(&$vars, $hook) {          // Nastavim do premennych zvlast teaser a zvlast body         // Znova nahram node pretoze <!--break--> uz bol vyfiltrovany         $node = node_load($vars['nid']);         if ($vars['page'] == TRUE) {                 if (strpos($node->body, '<!--break-->') !== 0) {                         $vars['style_teaser_differently'] = TRUE;                         $vars['node_teaser'] = check_markup($node->teaser, $node->format, FALSE);                         $vars['node_body'] = substr($node->body, strpos($node->body, '<!--break-->'));                 }         }         else {                 $vars['node_teaser'] = check_markup($node->teaser, $node->format, FALSE);         }    }

a potom v tvojej template, pouzijes na vypis teaseru normalne:

<div class="article-teaser"><?php print $node_teaser; ?></div> <div class="article-body"><?php  print $node_body; ?></div>

To je moj postup, neviem ci existuje lepsi a „cistejsi“.
