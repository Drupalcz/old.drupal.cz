Zjednodušeně narychlo nějak

Napsal uživatel Eva Rázgová dne 3 Červen, 2010 - 15:31.

Zjednodušeně narychlo nějak takhle:

$image_alt = $node->co potřebuju vypsat
$image_title = $node->co potřebuju vypsat
$image_filepath = $node->moje_pole_s_obrazkem[0]['filepath'];
$image_filename = $node->moje_pole_s_obrazkem[0]['filename'];
$image_preset = 'nazev_presetu';
$image = theme('imagecache', $image_preset, $image_filepath, $image_alt, $image_title);
$image_pres_lightbox = l($teaser_image, $image_filepath, array('attributes' => array('rel'=>'lightbox[group-nepovinne]['caption-nepovinne']',), 'html'=>TRUE));

Pozor: Při výpisu jakýchkoliv dat pocházejících od uživatelů na bezpečnost: najdětě si, k čemu je a jak se používá check_plain(), a dále klíčová slova Drupal + secure + code

V HTML by mělo stačit dát

Napsal uživatel Eva Rázgová dne 4 Červen, 2010 - 07:01.

V HTML by mělo stačit dát rel=„lightbox“ (podívejte se na README soubor modulu Lightbox). Nicméně, mít někde v šabloně, případně v uzlu natvrdo img src = (ještě s pevnou velikostí) není principiálně dobře, rozhodně pokud podobně zpracovaných obrázků na webu bude víc. K ošetření velikosti je právě lepší modul Imagecache a vypisovat to přes něj (což jsme Vám radili).
