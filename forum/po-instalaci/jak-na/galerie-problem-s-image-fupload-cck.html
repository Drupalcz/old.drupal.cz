Galerie - problém s Image FUpload (CCK)

Vložil/a Gynekolog, 24 Září, 2010 - 19:07
Kategorie: Jak na to?
Týká se verze: Drupal 6.x

Po celym promarněnym dni kdy sem skoro nic nevykoumal sem se rozhod zase otravovat ostatní. Snažim se zprovoznit galerii, naprosto jednoduchou..chci zvolit několik obrázků najednou ať se nahrajou a zobrazí všechny za sebou. Začal sem tak, jak by se asi začít mělo. Nainstaloval sem cck, imagefield, imagecache, image fupload (do kterýho sem stahnul ty soubory co sou potřeba). Vytvořil sem nastavení v imagecache pro zobrazení obrázku a pro náhled. Dál vytvořil typ obsahu Galerie kam sem přidal field Type → Image FUpload. Nastavil sem abych měl všechny obrázky na jedný stránce a abych moh přidávat neomezeně souborů, uložil. Všechno vypadalo že funguje, dal sem vytvořit Galerie, kliknul na zelený tlačítko, vybral fotky, klik na upload images, proběhlo (obr. 1).Pokračuju kliknutím na next step a první error. Fotka se nezobrazila (obr. 2). Nic, pokračoval sem kliknutím na „Done Editing“ a hodilo mi to bílou obrazovku s nápisem „Fatal error: Call to undefined function imagefield_con­tent_is_empty() in /drupal/defau­lt/modules/cck/con­tent.module on line 935“ hledal sem na googlu a našel http://drupal.org/node/541768. Z toho sem pochopil že bych měl změnit obsah na řádku 363 a 364 v souboru content.node_for­m.inc na
// Filter out items flagged for removal.
if ($field) {
$items = content_set_em­pty($field, $items);
}

tim sem se zbavil chybový hlášky. Místo ní mi to ukáže to co je na obr.3. Takže mi to napíše že je všechno ok ale obrázek nikde. Když to házelo chybovou hlášku, vyjel sem pryč tak všechno bylo uloženo ale místo obrázku mi to ukázalo to samý jako na obr.2.

Napsal uživatel karlos dne 3 Říjen, 2010 - 07:24.
  1. nakopirovat ho do vlastniho theme
  2. nakopirovat ho znovu do vlastniho theme se jmenem content-field-field_image.tpl­.php (tucne nahradte vlastnim nazvem obrazku)
  3. zmente kod nejak takhle:
<?php if (!$field_empty) : ?>
<div class="field field-type-<?php print $field_type_css ?> field-<?php print $field_name_css ?>">
  <?php if ($label_display == 'above') : ?>
    <div class="field-label"><?php print t($label) ?>:&nbsp;</div>
  <?php endif;?>
  <div class="field-items">
    <?php $count = 1;
    foreach ($items as $delta => $item) :
      if (!$item['empty'] && (($teaser == 1 && $count < 7) || $teaser == 0)) : ?> //tady ovlivnite pocet tech fotek ktere se zobrazi v teaseru - aktualne 6
        <div class="field-item <?php print ($count % 2 ? 'odd' : 'even') ?>">
          <?php if ($label_display == 'inline') { ?>
            <div class="field-label-inline<?php print($delta ? '' : '-first')?>">
              <?php print t($label) ?>:&nbsp;</div>
          <?php } ?>
          <?php print $item['view'] ?>
        </div>
      <?php $count++;
      endif;
    endforeach;?>
  </div>
</div>
<?php endif; ?>

snad jsem tam takhle po ranu neseknul nejakou chybu v te podmince.
